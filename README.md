# 🌏 Cambo Trip

**Cambo Trip** is a full-stack travel web application built with Laravel that helps users discover beautiful destinations across Cambodia. The platform provides destination information, AI-powered travel assistance, search functionality, and an admin dashboard for managing tourism content.

---

## ✨ Features

* 🗺️ Browse popular tourist destinations in Cambodia
* 🔍 Search destinations by name and category
* 🤖 AI-powered travel assistant
* 📍 Google Maps integration
* ❤️ Save favorite destinations
* 📱 Responsive design for desktop and mobile
* 🔐 User authentication
* 👨‍💼 Admin dashboard for content management
* 🖼️ Dynamic image management using image URLs

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/AydenzzVD/Cambo-Trip.git
```

Go into the project directory

```bash
cd Cambo-Trip
```

Install PHP dependencies

```bash
composer install
```

Install JavaScript dependencies

```bash
npm install
```

Create the environment file

```bash
cp .env.example .env
```

Generate the application key

```bash
php artisan key:generate
```

Configure your database inside `.env`.

Run the migrations and seeders

```bash
php artisan migrate --seed
```

Start the development server

```bash
php artisan serve
```

Run Vite

```bash
npm run dev
```
